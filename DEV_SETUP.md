## Developer Setup Guide ##

This file documents how to set up and start the project locally.

This avoids forgetting steps in the future.

⸻

Step 1 — Clone Repository

git clone <repo-url>
cd news-posts-app


⸻

Step 2 — Project Structure

backend/
frontend/
README.md
DEV_SETUP.md


⸻

Step 3 — Git Workflow

Check status: git status
Add files: git add .
Commit: git commit -m "message"
Push: git push

⸻

Notes :

Git does not track empty folders.
Use .gitkeep files to track folder structure.

⸻

Project Goal

Learn full-stack development by building a News Publishing Application using:
	•	React frontend
	•	FastAPI backend
	•	PostgreSQL database
